DoCode

A tool that automatically generates media files, such as videos, GIFs, and screenshots, from your JavaScript sketch file.

Installation (on a Mac)

  1. Make sure that Node.js is installed on your machine:
    • Open your terminal and type:
      $ node -v
      => v6.9.4 (or higher)
    • If Node.js is not installed, please go to the Node.js official site and install it.
  2. Make sure that ImageMagick is installed on your machine:
    • Open your terminal and type:
      $ convert
      => Should return the ImageMagick instructions.
    • If ImageMagick is not installed, please go to the ImageMagick download page and install it, or or type the following command on the terminal:
      $ brew install imagemagick
  3. Install the DoCode NPM package globally:
    • $ sudo npm install DoCode -g

      You might be asked to enter you Mac user password.
  4. Check that DoCode was installed correctly:
    • $ docode

      Should not return anything ‚úä.

How to use

  • Generate screenshots for a p5.js sketch
    • While in the project main folder, run the following command:
      $ docode --screenshots
      or
      $ docode -s
      • A new folder with the screenshots will be created at the same location.

    • When not on the project main folder, you can specify the project folder like this:
      $ docode --screenshots=[path_to_project_main_folder]
  • Generate a GIF for a p5.js sketch
    • While in the project main folder:
      $ docode --gif
      or
      $ docode -g
      • A new folder with the screenshots will be created at the same location.

    • When not on the project main folder:
      $ docode --gif=[path_to_project_main_folder]
  • Generate a video for a p5.js sketch
    • While in the project main folder:
      $ docode --video
      or
      $ docode -v
      • A new folder with the screenshots will be created at the same location.

    • When not on the project main folder:
      $ docode --video=[path_to_project_main_folder]

Uninstall (on a Mac)

  1. Use the following command:
    $ sudo npm uninstall -g docode

    You might be asked to enter you Mac user password.
