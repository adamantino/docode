doCode

A tool that automatically generates media files, such as videos, GIFs, and screenshots, from your JavaScript sketch file.

Installation

  1. Made sure that Node.js is installed on your machine:
    • Open your terminal and type:
      $ node -v
      => v6.9.4 (or higher)
    • If Node.js is not installed, please go to the Node.js official site and install it.
  2. Install the doCode NPM package globally:
    • $ npm install docode -g
  3. Check that doCode was installed correctly:
    • $ docode
